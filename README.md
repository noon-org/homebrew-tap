# Noon Analytics Homebrew Tap

Homebrew formulae for [Noon Analytics](https://noon.sh) tools.

## Installation

```bash
brew tap noon-org/tap
brew install finetype
```

Or install directly:

```bash
brew install noon-org/tap/finetype
```

```bash

```
